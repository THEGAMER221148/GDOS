Welcome to GedagadegadOS!
You are currently in the TERMINAL. Here is some useful information to get you started:
  • Type by pressing keys on your keyboard. If you can't type, try clicking the screen.
  • GDOS has little-to-no graphical interface, so you will have to navigate primarily using Gedagadegadagadacode in the terminal. Type "/help:;" for more information on how to code in the language.
  • All locally-stored data is kept within a Javascript object. Type "/dir:help:;" to learn more about GDOS's data storage.
  • You can upload files, such as GCode programs, using "/import:" followed by your desired upload location.
  • You can download files of any extension using "/export:" followed by the desired extension of the file and the location of the data you want to export.
  • You can run programs using "/run:program:" followed by the path to your desired program

Whatever you decide to use this joke of a programming language for is up to you, but I hope you enjoy it!
